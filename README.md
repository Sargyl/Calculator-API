# Калькулятор API

Веб-приложение калькулятора с REST API на Flask.

##  Функциональность

-  Арифметические операции (+, -, *, /)
-  Возведение в степень (^)
-  Квадратный корень (√)
-  Факториал (!)
-  История операций
-  Веб-интерфейс
-  REST API
